1. Download ResellPortfolio4 at https://github.com/hourzn/4630f2021
2. Make sure node / npm is install on your computer. https://nodejs.org/en/
3. Type in npm i to install and then once finished, type in "npm run start" to start the project and wait till QR code shows up,
4. Next Scan QR code with phone or run an emulator. 
5. If you scanning with QR code, you need to download EXPO GO app on the app store and create an account.
6. Once scanned, wait for bundle to finish.
7. Done.
